# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please email security@looppay.xyz

Do not open public issues for security vulnerabilities.

## Audit Status

This project has not been audited yet. Use at your own risk.

## Security Features

- UUPS upgradeable pattern with owner-only authorization
- ReentrancyGuard on payment execution
- Role-based access control for executors
- Custom errors for gas optimization
